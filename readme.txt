Removes text styling from blogs displaying black text on white background.

== Description ==
Javascript temporarily removes all background color and images from a blog page. Increases readability.

Useful for vision impaired users on cluttered or low contrast themes.

Useful for debugging themes.

The effect is temporary and goes away on the next page or when the page is refreshed.

It does not affect anything except background and text color.

== Installation ==

1. Download the plugin.
2. Upload the plugin to your wp-content/plugins directory.
3. Activate the plugin (all configuration is done through the widget).
4. Drag the widget to a sidebar.
5. Edit the sidebar title if needed.
6. Edit the link Text
10. Save
